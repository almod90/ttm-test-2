# Task 2

Создать объект (с любым набором свойств, но не менее 3-х разных свойств) и отправить его на сервер в
JSON-формате.

## Запуск

- `npm install`
- `npm run build`
- `npm run live`
- Open http://localhost:9000

## Описание
- Кликнуть кнопку, объект будет отправлен на сервер
